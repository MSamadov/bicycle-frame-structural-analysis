# Bicycle Frame Structural Analysis

## Project Overview
This project presents the design and structural analysis of a bicycle frame created using Autodesk Fusion 360.

The goal was to evaluate the structural performance of the frame under rider weight and loading conditions.

## Tools Used
- Autodesk Fusion 360
- Static Stress Simulation

## Design Description
The bicycle frame was modeled using parametric CAD techniques.

Key considerations:
- weight distribution
- structural rigidity
- load transfer through frame joints

## Simulation
A static structural analysis was performed to evaluate:

- stress distribution
- deformation
- structural safety

## Results
The simulation results indicate that the frame structure can sustain the expected loading conditions with acceptable deformation levels.
